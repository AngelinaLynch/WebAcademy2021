Inline CSS:

Inline CSS is when you put CSS into a html to alter it.This style sheet allows you to modify a single element into a unique one.The style sheet is fit into the head using "stylesheet". It can change your background color, your font color and so on. In like CSS is different from regular CSS because "it allow you to apply style rules to specfic HTML elements".

Internal CSS

Internal CSS is used to define a style for a SINGLE html page. Its found in the head aswell as any other CSS. Internal CSS changes the whole page into whatever color/font or type of style your looking for. Unlike external CSS its only for one page and isnt gonna change your whole website. Internal CSS is good if your doing a different theme a page or if you just want to be more creative with your work.

External CSS

Just like All of these different CSS types this one also changes the pages. BUT it doesnt just change one it changes all the pages for the website. For this to work you need to use the correct link in the head. This will allow you to change your whole look of your website with only one link!
